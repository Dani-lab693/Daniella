# Portfolio Website - Daniella

A modern, fully responsive portfolio website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Mobile-first approach that works seamlessly on all devices
- **Smooth Navigation**: Sticky navigation bar with smooth scrolling between sections
- **Hero Section**: Eye-catching landing section with call-to-action button
- **About Section**: Personal information with statistics
- **Skills Section**: Organized display of technical skills
- **Projects Showcase**: Featured projects with descriptions
- **Contact Form**: Functional contact form with validation
- **Footer**: Links and social media connections
- **Animations**: Smooth scroll animations and hover effects
- **Mobile Menu**: Hamburger menu for mobile devices

## Sections

1. **Navigation Bar** - Sticky header with navigation links
2. **Hero Section** - Introduction and call-to-action
3. **About** - Personal bio and statistics
4. **Skills** - Technical skills organized by category
5. **Projects** - Featured portfolio projects
6. **Contact** - Contact form for inquiries
7. **Footer** - Footer with links and information

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with gradients, flexbox, and grid
- **JavaScript (Vanilla)** - Interactive features and animations

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. Customize the content with your own information

## Customization

To personalize this portfolio:

- **Edit text content** in `index.html`
- **Modify colors** in the `:root` section of `styles.css`
- **Update project information** in the Projects section
- **Add your social media links** in the footer
- **Customize skills** in the Skills section

## Browser Support

- Firefox (latest)
